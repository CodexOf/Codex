Основная информация

Codex - это открытая база знаний по программированию и смежным темам.

Цели проекта

  • Создание структурированной базы знаний
  • Доступное изложение сложных концепций
  • Поддержка сообщества разработчиков

Технологии

Проект использует:

  • Markdown для контента
  • MkDocs для генерации сайта
  • Git для контроля версий
